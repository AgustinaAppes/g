¿Qué es Bootstrap?


Este es el mensaje secreto del punto 8.

Bootstrap es una de las librerias mas populares para el diseño y la maquetación web.

Incluye plantillas de diseño basadas en HTML y CSS para la realización de formularios, botones, tablas, navegación… así como complementos de JavaScript opcionales para mejorar el comportamiento de estos elementos. Esto te permite conseguir una web responsive en los navegadores modernos actuales invirtiendo el menor tiempo posible en maquetación.


Ventajas de Bootstrap


  • Fácil de usar: si tienes conocimientos básicos de HTML y CSS puede comenzar a usar Bootstrap cuando quieras.
  • Responsive: gracias a sus CSS tu diseño se adaptará a cualquier dispositivo.
  • Multinavegador: Bootstrap 4 es compatible con todos los navegadores modernos (Chrome, Firefox, Internet Explorer 10+, Safari y Opera).

Componentes principales


  1. Grid
    El trabajo en cuadrícula es la propiedad más importante que tiene Bootstrap, ya que nos resuelve el problema de visionado de elementos en diferentes resoluciones y dispositivos. Grid permite definir comportamientos determinados a cada espacio y características de pantalla. Para solucionar el problema del reparto, Bootstrap divide la pantalla en una cuadrícula de 12 columnas. Para que después tú mismo puedas gestionar esas columnas según el tamaño de la pantalla en cuestión.
  2. Navegación
    Un menú o barra de navegación es un encabezado que se coloca en la parte superior de la página.
  3. Botones
    Los botones son uno de los elementos más importantes de tu web. Bootstrap por defecto te proporciona clases para realizar diferentes botones según su cometidos y estado:
    • Botones semánticos: botones con significado por sí mismo en su color. Rojo significa alerta, amarillo cuidado, etc.
    • Botones contorneados: perfectos para acciones secundarias.
    • Botones con estado: activado, desactivado, etc.
    • Botones con tamaño: grandes, pequeños, de bloque, etc.
